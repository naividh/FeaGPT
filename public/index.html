Overview

FeaGPT is an end-to-end agentic AI system for Finite Element Analysis. It implements the Generate-Mesh-Solve-Analyze (GMSA) pipeline that automates the entire FEA workflow.

The system combines LLM reasoning with domain-specific engineering knowledge to handle geometry generation, meshing, solver configuration, and result interpretation.

Architecture

Planning Module    - LLM-based analysis planning
Geometry Engine     - Parametric geometry with validation
Meshing System      - Adaptive mesh generation
Simulation Core     - FEA solver interface
Analysis Suite      - Post-processing, fatigue, optimization
Batch Processing    - Parallel execution

GMSA Pipeline

Generate: Interprets problem description, creates parametric CAD model.

Mesh: Automatic mesh generation with adaptive refinement at stress concentrations.

Solve: Assembles stiffness matrix, solves with direct/iterative methods.

Analyze: Extracts stress fields, displacements, safety factors. Fatigue life prediction.

API Reference

POST /api/analyze

{
  "description": "Cantilever beam, 500mm, Al-7075, 1000N tip",
  "analysis_type": "static",
  "mesh_density": "medium"
}

GET /api/results/{job_id}

Returns stress, displacement, safety factor results.

GET /api/materials

Returns full materials database.

Examples

import feagpt

pipeline = feagpt.GMSAPipeline()
result = pipeline.run(
    description="Cantilever beam, 500x50x30mm",
    material="Al-7075-T6",
    load={"type":"point","magnitude":1000,"direction":"-Y"}
)
print(f"Max stress: {result.max_stress:.1f} MPa")
print(f"Safety factor: {result.safety_factor:.2f}")
