Overview FEA Solver Elements API Reference

Overview

FeaGPT v1.0 implements a real Finite Element Analysis solver entirely in JavaScript. Unlike simplified analytical tools, this performs actual stiffness matrix assembly, numerical integration, and iterative solving.

Key capabilities: Hex8/Tet4 elements, Gauss quadrature integration, sparse PCG solver with Jacobi preconditioner, real mesh generation, von Mises stress computation from actual strain-displacement relations.

FEA Solver Architecture

1. Mesh Generation    - Structured hex/tet mesh from parametric geometry
2. Element Stiffness  - 8-point Gauss quadrature for Hex8, 1-point for Tet4
3. Global Assembly    - Sparse COO format, assembled via connectivity
4. Boundary Conditions - Penalty method (K_ii * 1e20)
5. Load Application   - Consistent nodal forces from surface tractions
6. PCG Solver         - Preconditioned Conjugate Gradient with Jacobi precond
7. Post-Processing    - B-matrix strain extraction, D-matrix stress, von Mises

Element Library

Hex8: 8-node trilinear hexahedral element. Shape functions: Ni = (1/8)(1+xi_i*xi)(1+eta_i*eta)(1+zeta_i*zeta). 2x2x2 Gauss integration. Full 24x24 element stiffness matrix.

Tet4: 4-node linear tetrahedral element. Constant strain element. B-matrix computed from shape function derivatives. 12x12 element stiffness matrix.

API Reference

FEAEngine.solve(mesh, material, bcs, loads)

Returns: {
  displacements: Float64Array,  // nodal DOF values
  stresses: Float64Array[],     // element von Mises
  strains: Float64Array[],      // element strains [6]
  iterations: number,           // solver iterations
  residualNorm: number,         // final residual
  converged: boolean
}
